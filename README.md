**Рандомайзер аниме AniDrop**
___
Это приложение-рандомайзер для выбора аниме к просмотру.  
Сначала вы составляете пул тайтлов, добавляя их по сезону (например, зима 2026) или же по вашему списку запланированного на сайте Шикимори, предварительно авторизировавшись там через данный сервис.  
Затем вы добавляете тиры аниме, выставляя им шансы выпадения и распределяете тайтлы по этим тирам. После окончания распределения можете начинать ролить ваше аниме.  
___
Для того, чтобы запустить сервис локально, вам необходимо запустить базу данных, ввести необходимые параметры в файле [appsettings.json](./appsettings.json) в строке "DefaultConnection", придумать любой секретный ключ в строке "Secret" секции "JwtSettings". Для получения доступа к вашему списку Shikimori, необходимо зарегистрировать приложение на [данной](https://shikimori.io/oauth/applications) странице, а затем ввести полученные данные в секции "ShikimoriSettings" того же файла. В качестве "RedirectUri" указывайте адрес, на котором запускается ваше приложение.
___
В файлах [index.html](./wwwroot/index.html) задать значение переменной **API_BASE_URL** (строка 320) равной вашему адресу (указать порт, на котором запускается ваше приложение). Тот же адрес указать в файле [Program.cs](./Program.cs) в строке 55 и файле [ShikimoriController.cs](./Controllers/ShikimoriController.cs) в строке 49.
